# v0.2.0 — A 阶段稳定性 + B 阶段还债 + C 阶段功能 + Windows 安装器

> 一轮 14 个 commit 的整体迭代。本地阅读器在「稳、快、好用」三件事上都往前推了一步。

## 🛡️ A — 稳定性补强 (评审里指出的 P0)

- **RAG 检索释锁**：把 `search_chunks` 拆成"锁内 fetch chunks + 锁外 spawn_blocking 算余弦"，滚动 + 同时问 AI 不再卡顿
- **EPUB / MOBI HTML 白名单清洗** (`ammonia`)：堵死 `<script>` / `onerror` / `javascript:` URL / `<iframe>` / `<form>` / `style` 注入面
- **API key 走 OS Keychain** (`keyring`)：Windows Credential Manager 存密钥，DB 文件里不再有明文，备份 / 同步盘不再泄密
- **进度按段落锚定**：`reading_progress` 加 `paragraph_index + char_offset`，字号 14→20 切换后进度不再漂
- **tracing 三件套**：`%APPDATA%\com.aireader.app\logs\aireader.log` 滚动 JSON 日志，关键路径打 span（TTFB / 总耗时 / 锁等待）

## ⚙️ B — 内部债处理

- **`book_chunks` 加 `embedding_dim` / `embedding_model` 列**：未来换模型时不会静默失效
- **DB 全局 Mutex 换 `r2d2` 连接池** (max=8)：RAG 不再阻塞 `save_progress` / `add_read_time` 高频 IPC
- **EpubView 1504 → 1307 行** (-197)：拆 7 个独立 hook (Fullscreen / ReaderSettings / BookmarksPanel / ReaderKeybindings / SelectionPopover / ChapterEntities / ReadingProgress)

## 🎁 C — 新功能

- **Ctrl/Cmd+K 命令面板** — 全局模糊匹配跳书 / 视图 / 书签 / 设置
- **全文 FTS5 检索** — 中文友好的 trigram 分词，跨所有已索引的书秒回，命中片段高亮
- **二阶段 RAG**（cosine + BM25 RRF 融合）— 中文短查询召回明显更准
- **Calibre 库直连导入** — 选 Calibre 库目录一键把元数据 + 封面 + 文件路径都导入
- **高亮导出为 EPUB** — 后端 zip 打包，Apple Books / Calibre 可直接打开看
- **AI 对话沉淀为笔记** — ChatPanel 满意的回答一键存到 NotesView 的「AI 笔记」tab
- **读书日历** — 月历视图，每天热度色块 + 点开看当日阅读时长 / 高亮 / 书签
- **近 30 天阅读柱图** — StatsView 顶部，周末浅色，一眼看阅读节奏
- **推荐反馈闭环** — 推荐卡 hover 出现 [👎不感兴趣 / 📚待读 / ✓已完成]，dismissed 的不再出现

## 💾 下载

## 系统要求

- Windows 10 / 11
- WebView2（Win11 自带；Win10 装过 Edge 也有；缺失时安装器会自动提示）

## 已知提示

- **SmartScreen 拦截**：未做 code signing，首次启动点「更多信息 → 仍要运行」
- **首次 AI 索引会下载 ~120 MB 嵌入模型**（`BAAI/bge-small-zh-v1.5`），一次性，缓存在 `%APPDATA%\com.aireader.app\embed_cache\`
- **数据目录**：`%APPDATA%\com.aireader.app\`（db / 封面 / 日志 / 嵌入模型 / 音乐缓存都在这里）
- **API key 不在 db 文件里**：备份 db 不会泄密；换电脑时设置面板里重新填一次

## 升级

老版本直接装新版会自动跑两个迁移（幂等）：
1. SQLite 里残留的明文 `api_key` 迁到 OS Keychain，DB 字段清空
2. `book_chunks` 表加 `embedding_dim` / `embedding_model` 列，老数据按 `BGESmallZHV15 / 512` 回填

迁完日志里能看到 `已把旧版 SQLite 里的 api_key 迁移到 OS keystore`。
